## Components Used

### Hardware
- ESP32 Development Board
- Single-Channel Relay Module (5V, opto-isolated)
- AC Bulb with Holder
- Connecting Wires (Jumper Wires)
- USB Cable (for ESP32 power)

### Software
- Blynk IoT Mobile Application
- Arduino IDE
- Blynk ESP32 Library

### Tools
- Digital Multimeter
